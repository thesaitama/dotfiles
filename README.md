# thesaitama dotfile

## System requirements

* bash 4.4

### Linux

* Debian / LHE

### Mac

* macOS 10.15, 11
* brew / MacPorts

### Windows

* Emacs configuration compatible with Windows 7-10

## How to install

``` bash
git clone https://github.com/thesaitama/dotfiles.git ~/dotfiles
cd ~/dotfiles
sh ./dotfileslink.sh
```

* for Windows Emacs user: simply run `setup-emacs.bat`
